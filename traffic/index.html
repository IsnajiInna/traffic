Traffic Project

This folder contains the PHP-based traffic project. Use the instructions below to view it on GitHub or run it locally.

Important files
`config.php`Configuration
`dashboard.php`Admin dashboard
`login.php`Login page
`register.php`Registration page
`style.css`Stylesheet
`traffic.sql`Database dump (note filename contains comma)
Create GitHub repo Run locally
Viewing on GitHub
  • GitHub will display the repository files and HTML previews, but it cannot run PHP. To show pages you must use a hosted PHP server.
  • To publish code to GitHub from PowerShell: 
    git init ; git add . ; git commit -m "Initial commit" ; git branch -M main ; git remote add origin https://github.com/USERNAME/REPO.git ; git push -u origin main
  • To share a live site, deploy to a PHP-capable host (e.g., Heroku (deprecated PHP support), Render, DigitalOcean App Platform, or a shared hosting provider). GitHub Pages does not run PHP.
Run locally (recommended)
  1. Open PowerShell in this folder: `cd \path\to\traffic`
  2. Start PHP built-in server (requires PHP installed): 
    php -S localhost:8000
  3. Open http://localhost:8000 in your browser. PHP pages will execute normally.
  4. If you use MySQL, import `traffic.sql` into your database and update `config.php` with DB credentials.
Filename note: The file `traffic,sql` includes a comma in its name. Consider renaming it to `traffic.sql` for compatibility when importing into tools or uploading.
